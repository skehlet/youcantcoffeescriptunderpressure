You Can't JavaScript Under Pressure

This is just a small tweak to @shovon's You Can't CoffeeScript Under Pressure, which I made accept Javascript.

Previous README:

A CoffeeScript version of Us Vs Th3m's "You Can't JavaScript Under Pressure".

Running

Just be sure to install Harp, and then run

$ harp server

And you should be able to see the app on http://localhost:9000.
